# IMDBMoviesDatabase
### This is an iOS challenge submission to CreditFins
## Features

- [x] Movies List showing movies with images and description, infinite scrolling and "Favourites" option.
- [x] Movies Details showing poster, description, reviews, rate and add to favourites option.
- [x] Available for offline use.

## Requirements

- iOS 14.4+
- Xcode 12.4

#### CocoaPods
Use [CocoaPods](http://cocoapods.org/) to install needed dependencies:

```ruby
pod install
```
